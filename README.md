# Galaxy Core — Multi Stage Universe

[▶️ Play Live](https://bhuvanbhuvi043.github.io/galaxy-core-game/)

A cinematic multi-stage cosmic experience where particles become galaxies, galaxies become super galaxies, and everything ends in a powerful Big Bang.

---

## Game Concept

- Stage 1: Collect particles → Charge core → Create galaxies
- Stage 2+: Repeat galaxy creation across multiple sectors
- Advanced stages: Galaxies evolve into super galaxies
- Final: All structures collapse into a Big Bang / Supernova

Each stage expands the scale of creation — from particles to the universe itself.

## Controls

- Drag on the canvas to attract current targets
- Hold Gravity to pull particles toward the core
- Pulse for a shockwave
- Space = Pulse
- P = Pause
- R = Restart
- M = Sound toggle
